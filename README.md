# Student Management System (Java) 🎓

A robust Command Line Interface (CLI) application built in Java to streamline the management of student records. This project demonstrates the application of Object-Oriented Programming (OOP) and efficient data handling.

## 📌 Problem Statement
Educational institutions often struggle with manual record-keeping, which is prone to errors and difficult to search. This project provides a digital solution to **automate student data management**, allowing administrators to efficiently create, retrieve, update, and delete student information in real-time.

## ✨ Features
- **Add New Students:** Capture essential details like Name, Roll Number, and Grade.
- **View Records:** Display a formatted list of all registered students.
- **Search Functionality:** Quickly find a student using their unique Roll Number.
- **Data Modification:** Update student details or remove records seamlessly.
- **Validation:** Basic checks to ensure data integrity during input.

## 🛠️ Technical Stack
- **Language:** Java (JDK 11 or higher)
- **Concepts:** Encapsulation, Abstraction, Inheritance, and Java Collections (ArrayList).
- **Environment:** Compatible with IntelliJ IDEA, Eclipse, or VS Code.
